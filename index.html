Project

Automated Food Quality Analysis Using Machine Learning and Image Processing

This project uses computer vision and machine learning to automatically check the freshness of fruits and vegetables. A YOLOv8 model detects and identifies food items in an image, and features like color and texture are analyzed using a machine learning classifier (e.g., Random Forest) to label each item as fresh or spoiled. The system includes a Tkinter-based GUI for easy image upload or live webcam testing, providing a fast and reliable alternative to manual inspection for quality control and food waste reduction.
